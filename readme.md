# React App

This is a simple React application.

## Getting Started

Follow the instructions below to set up and run the application on your local machine.
 * If you do not have npm installed, do so from [here](https://nodejs.org/en/download/package-manager).

### Installation

1. **Install dependencies:**

   ```bash
   npm i
   ```

### Running the Application

To start the development server and run the application, use the following command:

```bash
   npm start
```

This will start the application and open it in your default web browser. If it doesn't open automatically, you can access it at [http://localhost:3000](http://localhost:3000).
